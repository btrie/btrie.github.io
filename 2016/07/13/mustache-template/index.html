Mustache Template

語法極簡的 Template Engine，只有 9.29 KB，沒有 if, for 等語法，透過值判斷進行那種流程。
不只僅用在 HTML，codegen 程式語言也很合適，相當簡潔適合閱讀。

官網 mustache 列出所有支援的語言，各種流行語言都有函示庫支援。

file extension: mustache, mst

Tags

tag in template, 列出所有語法

Tag Syntax
Variable {{key}}
Unescaped Variable {{&key}} or {{{key}}}
Sections {{#key}} {{/key}}
Sections primitive context {{.}}
Inverted Sections {{^key}} {{/key}}
Comments {{! ignore me }}
Partials {{> other_template}}
Set Delimiter {{=<% %>=}}

Values

value in view, 類似 helper

Value Syntax
function value of Variable function() { return this.first + this.second }
lambda value of Sections () => (text, render) => render(text)

API

Variable

  • HTML escaped by default
  • unescape a variable: {{& name}} or {{{name}}}

CSS Sprites

使用 NodeJS 製作一張圖包含所有圖檔，修改各個圖檔成所需要的大小，並提供 CSS 存取單一圖檔。

當網站有很多圖檔，需要讀取多次才能下載所有圖檔，將所有圖檔包成一張圖，減少對伺服器請求次數。 以國旗為例, 將各國國旗轉成一張圖，利用CSS顯示單一國旗.

工具

  • lwip Light Weight Image Processor for NodeJS
    改變圖檔大小

  • sprity A image sprite generator
    產生單一圖檔與 CSS
